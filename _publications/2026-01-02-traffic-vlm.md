<div class="lang-en">

This project investigates how vision-language models behave on traffic-scene question answering tasks. The work covers dataset construction, evaluation protocol design, and bias analysis for multimodal models in road-scene understanding.

The repository centers on traffic VQA data processing, question-only control experiments, visual grounding with GroundingDINO, model attention analysis with Grad-CAM, occlusion-based diagnostics, and counterfactual case studies. The goal is to distinguish genuine visual reasoning from shortcut learning and dataset priors.

<div class="lang-zh">

该项目来自清华大学深圳国际研究生院陈鑫磊老师课题组实习，研究视觉语言模型在交通场景问答任务中的表现、偏差与可解释性。

项目内容包括交通 VQA 数据构建、评测协议设计、question-only 对照实验、GroundingDINO 视觉定位、Grad-CAM 注意力分析、遮挡实验以及反事实样本诊断。核心目标是区分模型是否真的理解图像，还是依赖问题文本、数据分布和场景先验完成回答。
