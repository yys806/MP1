**Module:** Web Projects

This private repository keeps two Discord bots and their deployment tooling. `shen-test` is a long-term assistant bot for conversation, reminders, user profiling, self-improvement, mem0 memory, ASR/TTS, web search, and workflow support; `shen-play` handles entertainment and lightweight interaction. A `web-admin` console manages configuration, model/API keys, files, runtime data, logs, and service restarts.

The repository also includes systemd service files, server bootstrap scripts, deployment tooling, and sanitized configuration templates so runtime secrets and cached data stay outside GitHub.

**模块：** 网页项目

这是一个私有 Discord 双机器人仓库，保留 `shen-test` 与 `shen-play` 两个机器人及其部署工具。`shen-test` 面向长期聊天、任务提醒、画像、自我进化、mem0 长期记忆、ASR/TTS、联网搜索和工作流协助；`shen-play` 负责娱乐与轻量互动。`web-admin` 统一管理配置、模型/API key、文件、运行数据、日志和服务重启。

仓库同时包含 systemd 服务文件、服务器初始化脚本、部署脚本和脱敏配置模板，运行密钥与缓存数据不会提交到 GitHub。
